# 🎵 Nakshatra's Music App

A responsive, web-based music player built using **HTML**, **CSS**, and **JavaScript**. This project offers a clean UI, basic playback controls, dynamic song listing, and enhanced features like volume control and drag-and-drop music upload.

---

## 🔥 Features

- 🎧 Play / Pause / Next / Previous functionality  
- 📃 Dynamic playlist with song names & cover images  
- 🔊 Volume Up / Down buttons  
- 📊 Real-time progress bar with seek support  
- 🖱️ Drag-and-drop support for adding your own `.mp3` files  
- 🎹 Keyboard shortcuts (Space, Arrow keys, Mute)  
- 📱 Responsive UI using custom CSS and Google Fonts  

---

## 🧩 Technologies Used

- **HTML5** – Structure of the player  
- **CSS3** – Custom styling with gradients and responsiveness  
- **JavaScript** – Functional logic for audio control and interactivity
